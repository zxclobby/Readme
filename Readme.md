# Программа управления квартирами

Программа управления квартирами разработана на языке C# с использованием объектно-ориентированной технологии разработки. Она предоставляет возможность создания квартир и добавления к ним комнат с указанием их назначения и площади.

## Описание классов

### Класс Apartment (Квартира)

Класс `Apartment` представляет собой модель квартиры и имеет следующие свойства:
- `Number` (Номер квартиры): номер квартиры
- `Owner` (Владелец): владелец квартиры
- `Rooms` (Список комнат): список объектов класса `Room`, представляющих комнаты в квартире.

Класс `Apartment` имеет следующие методы:
- `AddRoom(Room room)`: добавляет комнату в список комнат квартиры.
- `Show()`: отображает информацию о квартире.
- `ShowAll()`: отображает информацию о всех комнатах в квартире.

### Класс Room (Комната)

Класс `Room` представляет собой модель комнаты и имеет следующие свойства:
- `Name` (Название): назначение комнаты
- `Area` (Площадь): площадь комнаты

Класс `Room` реализует интерфейс `IComparable`, который позволяет сравнивать объекты комнат на основе их площади.

## Консольное приложение

Консольное приложение использует созданную dll-библиотеку и демонстрирует работу классов. Оно выполняет следующие действия:
1. Создает объект квартиры.
2. Добавляет несколько объектов комнат с произвольными данными.
3. Отображает информацию о всех комнатах в квартире, упорядоченных по площади.
